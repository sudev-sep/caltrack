AI-Powered Fitness & Nutrition Tracker- CALTRACK
A modern, full-stack web application that uses Google's Gemini AI to completely simplify calorie and exercise tracking. Instead of manually searching databases and entering portion sizes, users simply type what they ate or did in natural language, and the AI handles the rest.


Tech Stack
Frontend: Angular, TypeScript, HTML5, CSS3, Bootstrap 5

Backend: Python, Django, Django REST Framework (DRF).

AI Integration: Google Generative AI .

Database: SQLite (Development)
